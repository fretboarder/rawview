# Test Data Corpus

Raw image test files for RawView accuracy and format coverage testing.

## Source

Files are sourced from [raw.pixls.us](https://raw.pixls.us/) — a public repository of camera raw sample files.

## Structure

Each test file has a corresponding JSON sidecar in `reference/` containing:
- 10 known photosite coordinates with expected channel and u16 value
- Expected CFA pattern
- Expected dimensions
- Expected black/white levels

## Usage

```bash
# Run accuracy tests (requires test files to be present)
TEST_RAW_FILE=/path/to/file.cr2 cargo test --workspace

# Full corpus test (when files are downloaded)
cargo test --workspace
```

## Git LFS

Large binary files (raw images) should be tracked with Git LFS:

```bash
git lfs install
git lfs track "test-data/*.cr2" "test-data/*.cr3" "test-data/*.nef" "test-data/*.arw" "test-data/*.raf" "test-data/*.rw2" "test-data/*.orf" "test-data/*.pef" "test-data/*.dng"
```
